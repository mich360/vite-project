コマンドライン実行履歴
% node -v   v18.17.0     確認

% npm create vite@latest  　　Vite入れる

npm create vite@latest というコマンドで、Viteを用いる

  • npm は、Node.jsのパッケージ管理ツールで、JavaScriptのパッケージ（ライブラリやツールなど）をインストール・管理します。
  • create は、npm の標準スクリプトで、新しいプロジェクトを作成するために使用されます。
  • vite@latest は、Viteパッケージを最新バージョンでインストールする指定です。@latest は最新バージョンを意味します。
